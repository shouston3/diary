# Diary

For 2018 I would like a place to write my diary.

# Quick start

```bash
git clone https://github.com/shouston3/diary_boilerplate.git && cd diary_boilerplate
```

```bash
mix deps.get

mix ecto.create

cd assets && npm run build && cd -

mix phx.start
```
